# My Typing Tutor

A TUI-based typing tutor for daily practice using random Wikipedia summaries.

## Preview

```

├─────────────────────────────────────────────────────────────────────────────┤

│ The mountain is located in the Cascade Range of the Pacific Northwest…      │
│                                                                             │
│ WPM: 72.4                         Accuracy: 98.6%                           │
│                                                                             │
│ [ type here… ]                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
